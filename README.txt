# Run tests with JUnit :
	* ant
	* ant test

# Run tests by main :
	* ant test-bymain

# Other available action:
	# Clean : ant clean
	# Init : ant init (depends on clean)
	# Compile : ant compile (depends on init)
	# Docs : ant docs (depends on compile)
	# Compile tests : ant compile-test (depends on compile)

	# Execute : ant exe | Does nothing for the moment. Use "ant test" or "ant test-bymain".
